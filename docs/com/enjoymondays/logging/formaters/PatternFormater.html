Classpublic class PatternFormater
InheritancePatternFormater AbstractFormater Object

Language Version : ActionScript 3.0

PatternFormater is used to format the message that gets logged by a particular Target. Inbuilt patterns: %d : Date of the log. %F : File of the log, uses StackTrace. %p : The log message. %C : Class. %M : Method name %m : Message. %n : New line. %L : Line number.
